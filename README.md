## Electron Weather Tray (TypeScript)

Простое приложение на **Electron + TypeScript**, которое показывает текущую температуру в **иконке в системном трее** и обновляет её **раз в минуту**.

Используется бесплатное API `open-meteo.com` (без ключа).

### Установка

```bash
cd /home/foobar/r/sandbox/cursor/weather
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка и запуск

```bash
npm run build
npm start
```

### Настройка города

По умолчанию используются координаты Москвы.
Измените константы `LATITUDE` и `LONGITUDE` в файле `src/main.ts` на свои:

```ts
const LATITUDE = 55.7558;
const LONGITUDE = 37.6173;
```

### Как отображается температура

- В **tooltip** при наведении курсора на иконку.
- В **тексте трея** через `tray.setTitle(...)` — полностью поддерживается в macOS, на Linux/Windows может быть проигнорировано средой рабочего стола.  
- В **контекстном меню** трея строкой `Текущая температура: ...`.
